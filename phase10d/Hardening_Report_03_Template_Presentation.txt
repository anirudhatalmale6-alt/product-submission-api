TEMPLATE LIBRARY CREATED

File: gutenberg_utils.py (canonical module)

New template functions added:

1. tpl_info_strip(effective_date, last_updated, website, business)
   Standard top-of-page metadata block showing dates and site info.

2. tpl_affiliate_disclosure()
   Standard affiliate disclosure banner using wp:html block with
   consistent styling (grey background, blue left border).

3. tpl_trust_footer(pages)
   Standard trust page cross-link section with separator, heading,
   and bulleted links to all trust/transparency pages.

4. tpl_faq_section(qa_pairs)
   FAQ section using wp:details blocks for collapsible Q&A.

5. tpl_related_reading(links)
   Related Reading section with separator, heading, and link list.

6. build_educational_post(...)
   Complete educational post builder: info strip -> quick summary ->
   body sections -> FAQ -> related reading -> trust footer.

7. build_trust_page(...)
   Complete trust page builder: info strip -> body sections ->
   trust footer.


DEFINED LAYOUT STANDARD
========================
All managed content follows this structure:

  For Educational Posts:
  1. Info strip (Last Updated, Website, Business)
  2. Separator
  3. Quick Summary (H2 + bold intro)
  4. Separator
  5. Body sections (H2/H3 hierarchy)
  6. FAQ section (if appropriate)
  7. Related Reading links
  8. Trust footer

  For Hub/Category Pages:
  1. Affiliate disclosure (if commercial)
  2. Info strip
  3. Cover/hero image
  4. Intro section
  5. Category grid / product sections
  6. FAQ section
  7. Trust/support block
  8. Related Reading

  For Trust/Policy Pages:
  1. Info strip
  2. Separator
  3. Body sections
  4. Trust footer (cross-links to all trust pages)
